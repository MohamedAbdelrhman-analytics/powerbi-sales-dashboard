# 📊 Power BI Sales Dashboard

![Dashboard](dashboard.png)

## Project Overview

This project presents an interactive Power BI dashboard built to analyze sales performance and provide business insights through dynamic visualizations.

## Features

- KPI Cards
- Revenue Analysis
- Profit Analysis
- Customer Analysis
- Product Analysis
- Monthly Sales Trend
- Interactive Filters

## KPIs

- Total Revenue
- Total Profit
- Total Orders
- Total Customers
- Profit Margin

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel
